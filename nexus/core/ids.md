# Stable IDs

This file records stable IDs for foundational NEXUS concepts.

## Why this file exists

Stable IDs make it easier to:

- link canonical docs
- generate site URLs
- attach forum discussions
- build tooling without fragile string matching

## Core IDs

- `nexus.node`
- `nexus.boundary`
- `nexus.context`
- `nexus.relation`
- `nexus.derivation`
- `nexus.lens`
- `nexus.domain`
- `nexus.meaning`
- `nexus.category`
- `nexus.kind`
- `nexus.role`

## Law IDs

## Guidance

- Prefer stable lowercase dot-separated IDs.
- Avoid renaming IDs once public.
- Human-readable labels may change; IDs should not.
